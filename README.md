# Clock Weather Switch

A GNOME Shell extension that replaces the standard clock in the top bar with a weather forecast at regular intervals. It smoothly animates between the current time and the local temperature/weather condition.

### Manual Installation
1. Download the source.
2. Copy the folder to `~/.local/share/gnome-shell/extensions/clock-weather-switch@erixx`.
3. Restart GNOME Shell (Log out/Login or `Alt`+`F2`, `r`, `Enter` on X11).
4. Enable the extension using **Extensions Manager** or **GNOME Extensions**.

## Configuration
Open the extension settings to:
1. Enter your city name (e.g., "Berlin", "New York").
2. Wait for the location to be detected.
3. Adjust the switch interval if desired.
